# Titanic Survival Analysis

## Overview

This project analyzes the Titanic dataset using Python, Pandas, Matplotlib, and Seaborn.

## Objectives

* Clean missing data
* Analyze passenger survival patterns
* Compare survival rates by gender
* Compare survival rates by passenger class
* Analyze survival rates across age groups
* Create visualizations for better insights

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Visualizations

* Survival Rate by Gender
* Survival Rate by Passenger Class
* Passenger Age Distribution

## Conclusion

The analysis shows that survival rates varied significantly by gender and passenger class. Female passengers and higher-class passengers had better chances of survival.
